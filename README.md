# tips

1. Alt+f12 触发 窗口透明，锁定，穿透。同样热键解除三种状态;
2. 将视频调整到全屏后，拖动右下角，缩放到合适大小位置再按Alt+f12；
3. Alt+F1快退视频10秒；
4. Alt+F2快进视频10秒；
5. Alt+·（破折号所在按键）暂停（开始）视频；
6. config.json可以修改以上快捷键，透明度等；
7. 该工具的开发python版本是3.8.2；
